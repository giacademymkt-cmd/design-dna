# Dev Tutorial: Demo + Code (`dev-tutorial-code-showcase`)

Familia dominante do perfil code.xr: reels educativos de front-end no formato 9:16 dividido em duas ou tres faixas horizontais. Em cima, um componente de UI real animado (upload, botao com barra de progresso, dock macOS, checkout fintech) flutuando com sombra suave sobre fundo claro/neutro; embaixo, uma janela de editor de codigo estilo macOS (tres bolinhas de semaforo, aba de arquivo, borda/label branded 'codeXR </>') com o snippet React/Framer Motion/Tailwind que gera o efeito. Ensina o resultado e o 'como' na mesma tela, formato altamente salvavel. Titulo bicromatico (uma palavra colorida) faz a hierarquia por cor, nao por tamanho.

**Quando usar:** Quando o objetivo e conteudo educativo de dev/design para Reels e carrosseis, documentacao de design system, ou demonstrar microinteracoes (upload, loading, dock, checkout) provando com codigo real. Use o combo demo-em-cima/codigo-embaixo e a janela de editor branded como assinatura.

## Tokens do estilo

- **Paleta assinatura:** `#FFFFFF`, `#0B0B14`, `#1E9BF0`, `#EC0F7B`, `#E07856`, `#C084FC`, `#4ADE80`
- **Tipografia:** Titulo em sans humanista/geometrica bold apertada, em duas linhas, com uma palavra em cor de acento (coral, magenta, azul) para enfase por cor. Codigo em monospace de editor com syntax highlight multicolor (keywords roxas, strings verdes, props azuis) e numeros de linha cinza. UI da demo em sans limpa tipo Inter/Geist.
- **Layout:** Vertical 9:16 empilhado: titulo centralizado no topo com respiro alto, palco claro com o componente centralizado no meio, e janela de codigo dark alinhada a esquerda ocupando a base. Densidade cresce de cima para baixo. Chrome macOS na janela de codigo com CTA discreto embutido ('Like & Follow for more', 'Code link in my bio').
